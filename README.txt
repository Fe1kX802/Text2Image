Text2Image - название программы (и проекта), test_text - текст для теста
!!Программа не умеет работать с русским текстом, только английский (ограничение 8 бит на цвет => 256 различных символов)!!
Txt2Img_window это последняя версия программы для работы в окне, а Txt2Img_console это версия той же самой программы для работы в консоли (над ней я еще работаю)
